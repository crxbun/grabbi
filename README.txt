# grabbi
LIBRARIES AND OPEN SOURCE PACKAGES USED:
Frontend: React
Backend: Flask
API: Spoonacular

HOW TO GET THEM:
1.	clone the repo
2.	cd into client folder
3.	run npm install
4.	run npm start to start the client side of the application
5.	cd back up one directory cd ..
6.	python3 run.py or simply run the run.py file to start the server
7.	locate localhost:3000 (client) and localhost:5000 (server) on your web browser
